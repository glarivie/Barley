## Barley

#### How to help ?

```
git clone git@github.com:hqro/Barley.git
cd Barley
yarn
yarn start
```

Use [Expo App](https://expo.io) to see your work

#### Where to download ?

You can try **Barley** just [here](https://exp.host/@hqro/barley)
